# 🧠 NeuroStudy AI

A Multi-Agent AI Learning Ecosystem built for the Google Cloud Rapid Agent Hackathon.

## What it does
Upload a PDF → 5 AI agents automatically generate notes, quizzes, study plans, revision tracker, and answer your doubts.

## Tech Stack
- Frontend: Next.js + Tailwind CSS
- Backend: FastAPI + Python
- AI: Gemini API
- Database: MongoDB
- Auth: Firebase
- Deployment: Vercel + Render

## Agents
- Notes Agent — generates summaries and key points
- Quiz Agent — creates MCQs with difficulty levels
- Planner Agent — builds a study timetable
- Doubt Agent — answers your questions
- Revision Agent — tracks weak topics
